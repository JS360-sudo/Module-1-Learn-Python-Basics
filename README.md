# 🐍 CyberVets Python – Module 1: Python Basics

Welcome to **CyberVets Python**, a self-paced beginner coding course designed for veterans, service members, and their families who are new to programming, cybersecurity, and computer science.

No prior coding experience is required. You just need curiosity and the willingness to learn.

---

## 📋 About This Module

**Module 1: Python Basics** introduces the foundational concepts of Python programming through short lessons, guided examples, and hands-on mini projects.

By the end of this module, you will be able to:

- Understand what Python is and how code works
- Create and use variables to store information
- Identify and work with basic data types
- Perform basic math operations and handle user input
- Write and run simple Python programs

## 🚀 How to Get Started

### Prerequisites

- Python 3.x installed on your computer
  - Download it here: [https://www.python.org/downloads/](https://www.python.org/downloads/)
- A code editor (recommended: [VS Code](https://code.visualstudio.com/))

### Running the Course

1. Clone or download this repository to your computer:
   ```bash
   git clone https://github.com/JS360-sudo/Module-1-Learn-Python-Basics.git
   ```

2. Open a terminal or command prompt and navigate to the folder:
   ```bash
   cd Module-1-Learn-Python-Basics
   ```

3. Run the intro file to begin:
   ```bash
   python Intro.py
   ```

4. Follow the on-screen prompts — the course will guide you through each lesson step by step.
