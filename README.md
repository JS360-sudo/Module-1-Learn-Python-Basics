# Module 1

This module covers some of the most common and basic commands in Python. Studens will work work through 3 lesson plans and simple program code at the end of the module to test their knowledge from what they learend during this module. Students will commit on completion of each file and push the conde when they are done with each lesson.

---

## Folder Structure

**Lesson 1:**
- `Lesson_1_Variables.py` - Individuals will learn some of the basic of variables in coding and how they work in Python.
  
**Lesson 2:**
- `Lesson_2_Data_Types.py` - Students will be introduced to common data types in python: Integers; floats; strings; booleans.

**Lesson 3:**
- `Lesson_3_Basic_Operations.py` - Students will be introduced to basic operations, inputs, and outputs. They will learn how to ask for user input; do basic math; and write comments

**Module 1 Mini-Projects:**
- `Simple_Programs.py` - Students will be challenged with creating a simple program from what they learned within the lessons of this module. 

---

## How to Use

1. **Start with the Module 1 folder:** Complete all the Lesson plans before attempting the "Mini_Project"
2. Open each file in your editor
3. Read the instructions at the top of the file
4. Write your solution in the designated area
5. Run the file to test your code
6. Commit after completing each file 

## Benefits of Individual Files

- **Focus** – Work on one concept at a time
- **Progress tracking** – Easily see which questions you've completed
- **Testing** – Run individual files without interference
